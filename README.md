# MUI Collapsible Sidebar React App

This is a React application using Material-UI with:

- Collapsible sidebar with icon-only mode
- Responsive layout using MUI components
- Routing using `react-router-dom`

## 📦 Setup

```bash
npm install
npm start
```

## 📁 Folder Structure

```
src/
├── App.jsx
├── index.js
├── layout/

├── components/

├── pages/

```
